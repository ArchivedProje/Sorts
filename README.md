# Sorts
| Name | Time Complexity| Memory Complexity |
|------|----------------|-------------------|
| Bubble Sort | O(N) -> O(N^2) -> O(N^2) | O(1) |
| Quick Sort | O(NlogN) -> O(NlogN) -> O(N^2) | O(logN) |
| Merge Sort | O(NlogN) -> O(NlogN) -> O(NlogN) | O(N) |
| Insertion Sort | O(N) -> O(N^2) -> O(N^2) | O(1) |
